## 概要

このタスクリストは、PDF RAG Pipeline レポジトリをMVPとして実装するための作業項目です。

目的は以下です。

- プロジェクトごとの `01_処理前` に置かれた PDF を自動検出する
- `opendataloader-pdf` で Markdown / JSON / HTML を生成する
- `02_処理後` に閲覧用 / RAG用 / 機械処理用として整理する
- JSON から RAG用チャンクを生成する
- README / manifest を自動生成する
